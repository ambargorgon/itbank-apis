1. OBTENER DATOS DE UN CLIENTE:
Un cliente autenticado puede consultar sus propios datos:

> http://127.0.0.1:8000/api/datos-cliente

2. OBTENER SALDO DE CUENTA DE UN CLIENTE: * (falta mostrar tipo de cuenta)
Un cliente autenticado puede obtener el tipo de cuenta y su saldo:

> http://127.0.0.1:8000/api/saldo-cliente

3. OBTENER MONTO DE PRESTAMOS DE UN CLIENTE
Un cliente autenticado puede obtener el tipo de préstamo y total del mismo

> http://127.0.0.1:8000/api/prestamo-cliente

4. OBTENER MONTO DE PRESTAMOS DE UNA SUCURSAL
Un empleado autenticado puede obtener el listado de préstamos otorgados de una sucursal determinada.

> http://127.0.0.1:8000/api/prestamo-sucursal/?sucursal_id=85

5. OBTENER TARJETAS ASOCIADAS A UN CLIENTE
Un empleado autenticado puede obtener el listado de tarjetas de crédito de un cliente determinado

> http://127.0.0.1:8000/api/tarjeta-cliente/?costumer_id=12

6. GENERAR UNA SOLICITUD DE PRESTAMO PARA UN CLIENTE
Un empleado autenticado puede solicitar un préstamo para un cliente, registrando el mismo y acreditando el saldo en su cuenta

> http://127.0.0.1:8000/api/solicitar-prestamo

7. ANULAR SOLICITUD DE PRESTAMO DE CLIENTE
Un empleado autenticado puede anular un préstamo para un cliente, revirtiendo el monto correspondiente

> http://127.0.0.1:8000/api/anular-prestamo/?loan_id=14&customer_id=353&loan_total=29782439

8. MODIFICAR DIRECCION DE UN CLIENTE
El propio cliente autenticado o un empleado puede modificar las direcciones

Admin > http://127.0.0.1:8000/api/modificar-direccion/?customer_id=10&directions=HOLAGUS
Cliente > http://127.0.0.1:8000/api/modificar-direccion/?directions=CHAUGUS

9. LISTADO DE TODAS LAS SUCURSALES
Un endpoint público que devuelve el listado todas las sucursales con la información correspondiente

> http://127.0.0.1:8000/api/sucursales
